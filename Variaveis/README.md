# Variáveis
Esta pasta reúne os primeiros exercícios desenvolvidos na disciplina, com foco no uso de variáveis e operações básicas.
Aqui estão atividades voltadas para entender como declarar, inicializar e utilizar variáveis em diferentes cálculos.

## O que foi trabalhado
* Declaração de variáveis
* Tipos primitivos (`int`, `double`, etc.)
* Operações aritméticas
* Atribuição e reatribuição de valores
* Entrada e saída de dados

## Objetivo
Construir a base da lógica de programação, compreendendo como os dados são armazenados e manipulados dentro de um programa.
