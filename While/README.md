# Estrutura de Repetição — While
Nesta pasta estão os exercícios realizados utilizando a estrutura de repetição `while`.
Os códigos aqui foram desenvolvidos para praticar a execução de comandos repetidamente enquanto uma condição for verdadeira, entendendo melhor como funciona o controle de fluxo em laços de repetição.

## O que foi trabalhado
* Uso do `while`
* Condições booleanas
* Contadores e acumuladores
* Validação de dados
* Controle de entrada pelo usuário

## Objetivo
Aprender a utilizar laços de repetição para resolver problemas que exigem execução contínua até que determinada condição seja atendida.
Esses exercícios ajudaram a reforçar a lógica e o entendimento do funcionamento de loops na prática.
