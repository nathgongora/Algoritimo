# Vetores
Esta pasta reúne os exercícios desenvolvidos utilizando vetores (arrays).
Os códigos aqui foram feitos para praticar o armazenamento e manipulação de múltiplos valores em uma única estrutura, entendendo como trabalhar com índices e percorrer posições na memória.

## O que foi trabalhado
* Declaração e inicialização de vetores
* Acesso por índice
* Percorrer vetor com laços de repetição
* Leitura e impressão de valores
* Cálculos utilizando elementos do vetor

## Objetivo
Compreender como organizar vários dados do mesmo tipo em uma única estrutura e como manipulá-los de forma eficiente.
