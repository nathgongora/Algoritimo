# If-Else
Esta pasta reúne os exercícios desenvolvidos utilizando estruturas condicionais `if` e `if-else`.
Os códigos aqui foram criados para praticar a tomada de decisão dentro do programa, permitindo que diferentes ações sejam executadas conforme determinadas condições.

## O que foi trabalhado
* Estrutura `if`
* Estrutura `if-else`
* Condições booleanas
* Operadores relacionais e lógicos
* Controle de fluxo

## Objetivo
Compreender como o programa pode reagir de maneiras diferentes dependendo das condições estabelecidas, desenvolvendo o raciocínio lógico para construção de decisões no código.
