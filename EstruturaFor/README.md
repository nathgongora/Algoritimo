# For
Esta pasta reúne os exercícios desenvolvidos utilizando a estrutura de repetição `for`.
Os códigos aqui foram feitos para praticar a execução controlada de repetições, especialmente quando já se conhece a quantidade de vezes que o bloco de código deve ser executado.

## O que foi trabalhado
* Estrutura `for`
* Controle de contador
* Inicialização, condição e incremento
* Laços aninhados
* Uso do `for` com vetores

## Objetivo
Entender como utilizar o `for` para tornar o código mais organizado em situações que exigem repetição com controle definido.
Esses exercícios ajudaram a reforçar o raciocínio lógico e o controle do fluxo em estruturas de repetição.
