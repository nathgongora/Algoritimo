# Switch Case
Esta pasta reúne os exercícios desenvolvidos utilizando a estrutura de decisão `switch case`.
Os códigos aqui foram feitos para praticar a tomada de decisão com múltiplas opções, tornando o fluxo do programa mais organizado quando 
há várias condições possíveis.

## O que foi trabalhado
* Estrutura `switch`
* Uso do `case` e `break`
* Estrutura `default`
* Comparação de valores
* Organização de menus simples

## Objetivo
Aprender a utilizar o `switch case` como alternativa ao `if-else` em situações com múltiplas escolhas, deixando o código mais claro e estruturado.
